# Meta-Ads-Intelligence-Dashboard-Campaign-Performance-Analytics-using-Power-BI
Interactive Power BI dashboard analyzing Meta (Facebook &amp; Instagram) ad campaign performance with KPIs like CTR, Conversion Rate, Engagement Rate, and Purchase Rate.

This project focuses on extracting actionable insights from campaign data by tracking key performance metrics, identifying trends, and evaluating ad effectiveness across different dimensions.

🚀 Project Objectives

Analyze 120K+ impressions across campaigns

Track key KPIs: CTR, Engagement Rate, Conversion Rate, Purchase Rate

Evaluate ad performance by:

Gender

Age Group

Country

Ad Type

Campaign Name

Monitor weekly and hourly impression trends

Compare performance across Facebook and Instagram

📈 Key Insights Covered

Total Impressions: 123K+

Clicks: 14K+

Engagement Rate: 13.6%

Conversion Rate: 4.82%

Purchase Rate: 0.57%

Total Budget Analyzed: $2.5M

🛠 Tools & Technologies Used

Power BI

DAX (Data Analysis Expressions)

Data Modeling

Data Cleaning & Transformation

KPI Design & Visualization

🎯 Features

Dynamic filtering by Campaign Name & Target Interest

Interactive KPI cards

Demographic analysis

Geographic performance mapping

Monthly and hourly trend analysis

Ad type comparison (Carousel, Image, Stories, Video)

💡 Business Impact

This dashboard enables marketing teams to:

Identify high-performing campaigns

Optimize budget allocation

Improve targeting strategy

Increase conversion efficiency

Make data-driven marketing decisions
